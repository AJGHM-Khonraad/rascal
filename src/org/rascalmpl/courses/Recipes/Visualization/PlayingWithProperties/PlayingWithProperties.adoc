.Examples
Here is an ellipse with minimum size 200x300 that occupies 80% of the available space:
[source,rascal-shell]
----

[source,rascal-shell]

----

[source,rascal-shell]

----

image::{concept}/e3.png[alt="e3"]

[source,rascal-shell]

----

image::{concept}/e4.png[alt="e4"]

[source,rascal-shell]
----
