# lang
:concept: lang

.Synopsis
Definitions and tools for various languages.

.Description


** <<lang-RSF,RSF>>: Functions for reading/writing the Rigi Standard Format (RSF).
*** <<RSF-readRSF,readRSF>>: Read a file in Rigi Standard Format (RSF).
** <<lang-csv,csv>>: Comma-Separated Values (CSV).
** <<lang-java,java>>: Definitions and tools related to the Java language.
*** <<java-jdt,jdt>>: tools for Java provided via reusing the Eclipse JDT.
*** <<java-m3,m3>>: Java-specific version of the M3 model.
** <<lang-xml,xml>>: XML utilities.
