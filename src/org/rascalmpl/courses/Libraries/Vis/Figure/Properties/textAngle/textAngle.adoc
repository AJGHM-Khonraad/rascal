# textAngle
:concept: Vis/Figure/Properties/textAngle

.Synopsis
Define the angle at which text is drawn.

.Syntax

.Types

.Function
`FProperty textAngle(num a)`

.Description
Place text at `a` degrees. The value of the angle may also be computed by a callback `ca`.
Also see <<Angles>>.
