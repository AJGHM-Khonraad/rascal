# top
:concept: Vis/Figure/Properties/top

.Synopsis
Top, vertical, alignment.

.Syntax

.Types

.Function
`FProperty top()`

.Description
`top` defines *top vertical alignment*.
It sets the vertical alignment to 0.

Note that

*  `top()` is equivalent to `valign(0.0)`.


Also see <<align>>, <<valign>> and <<CompositionModel>>.
