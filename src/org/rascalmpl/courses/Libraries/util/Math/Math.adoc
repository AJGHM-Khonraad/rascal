[[util-Math]]
# Math
:concept: util/Math
.Usage
`import util::Math;`



.Synopsis
Mathematical functions.

.Usage

.Description

The `Math` library provides the following functions:




* <<Math-E,E>>: The constant E.
      
* <<Math-PI,PI>>: The constant pi.
      
* <<Math-abs,abs>>: Absolute value of a number.
      
* <<Math-arbInt,arbInt>>: Generate a random integer value.
      
* <<Math-arbRat,arbRat>>: Generate an arbitrary rational value.
      
* <<Math-arbReal,arbReal>>: Generate a random real value in the interval [0.0,1.0).
      
* <<Math-arbSeed,arbSeed>>: Define the seed for the generation of arbitrary values.
      
* <<Math-ceil,ceil>>: Compute the smallest integer that is larger than a given number.
      
* <<Math-cos,cos>>: Calculate the cosine of a numeric value.
      
* <<Math-denominator,denominator>>: Return the denominator of a rational value.
      
* <<Math-exp,exp>>: Compute exp(x).
      
* <<Math-floor,floor>>: Compute the largest integer that is smaller than a given number.
      
* <<Math-ln,ln>>: Calculate the natural log of a numeric value.
      
* <<Math-log,log>>: Calculate the log<sub>base</sub> of a numeric value.
      
* <<Math-log10,log10>>: Compute the 10 based log(x).
      
* <<Math-log2,log2>>: Compute the 2 based log(x).
      
* <<Math-max,max>>: Determine the largest of two numeric values.
      
* <<Math-min,min>>: Determine the smallest of two numeric values.
      
* <<Math-nroot,nroot>>: Calculate the n<sup>th</sup> root of a numeric value.
      
* <<Math-numerator,numerator>>: Return the numerator of a rational value.
      
* <<Math-percent,percent>>: Compute the ratio between two numbers as a percentage.
      
* <<Math-pow,pow>>: Calculate an arbitrary power of a numeric value.
      
* <<Math-precision,precision>>: Return the precision of a real number.
      
* <<Math-primes,primes>>: generate prime numbers up to a maximum
      
* <<Math-remainder,remainder>>: Return the remainder of dividing the numerator by the denominator.
      
* <<Math-round,round>>: Round a number to the nearest multiple of a given number (default 1).
      
* <<Math-scale,scale>>: Return the scale of a real number.
      
* <<Math-setPrecision,setPrecision>>: Define the precision for numeric calculations; returns the previous precision.
      
* <<Math-sin,sin>>: Calculate the sine of a numeric value.
      
* <<Math-sqrt,sqrt>>: Calculate the square root of a numeric value.
      
* <<Math-tan,tan>>: Calculate the tangent of a numeric value.
      
* <<Math-toInt,toInt>>: Convert a numeric value to an integer.
      
* <<Math-toRat,toRat>>: Convert two numbers to a rational value (not supported on reals).
      
* <<Math-toReal,toReal>>: Convert a numeric value to a real.
      
* <<Math-toString,toString>>: Convert a numeric value to a string.
      
* <<Math-unscaled,unscaled>>: Return the unscaled integer of a real.

[[Math-arbInt]]
## arbInt

.Function 
* `int arbInt()`
          * `int arbInt(int limit)`
          


.Synopsis
Generate a random integer value.

.Description
Return an arbitrary integer value. When the argument `limit` is given, the generated value is in the interval [0, `limit`),
i.e., the limit is exclusive.

.Examples
[source,rascal-shell]
----
rascal>import util::Math;
ok
rascal>arbInt();
int: -13867586
rascal>arbInt();
int: 1079650598
rascal>arbInt();
int: 323809440
rascal>arbInt(10);
int: 5
rascal>arbInt(10);
int: 8
rascal>arbInt(10);
int: 9
----

.Benefits
`arbInt` is a convenient generator for pseudo-random integers.

[[Math-arbSeed]]
## arbSeed

.Function 
`void arbSeed(int seed)`


.Synopsis
Define the seed for the generation of arbitrary values.

.Description
Define the seed for the generation of arbitrary values such as <<arbBool>>, <<arbInt>>, <<arbReal>>,
<<arbRat>>, <<List-getOneFrom>>,<<Set-getOneFrom>>, <<List-takeOneFrom>> and <<Set-takeOneFrom>>. <<arbSeed>> resets the random number generator that

[[Math-toInt]]
## toInt

.Function 
`int toInt(num N)`


.Synopsis
Convert a numeric value to an integer.

.Description
Convert a number to an integer. If `n` is an integer, this is the identity. If `n` is a real value (implemented as BigDecimal) to an integer (implemented as BigInteger). This conversion is analogous to a narrowing primitive conversion from double to long as defined in the Java Language Specification: any fractional part of this BigDecimal will be discarded. Note that this conversion can loose information about the precision of the BigDecimal value.

.Examples
[source,rascal-shell]
----
rascal>import util::Math;
ok
rascal>toInt(13)
int: 13
rascal>toInt(13.5)
int: 13
----
